## Week 1: Data Engineering & Schema Design

### Objective
Prepare a clean, analytics-ready data model using a Star Schema for downstream RFM analysis.

### Dataset
- Retail transaction dataset
- 300+ unique customers
- Multiple purchases per customer

### Week 1 Tasks Completed
- Raw data understanding and validation
- Data cleaning checks (NULLs, blanks, invalid values)
- Date/time formatting validation
- Star Schema implementation:
  - Dim Customer
  - Dim Product
  - Fact Sales
- Creation of Single Customer SQL View for analytics

### Technologies Used
- MySQL
- SQL

### Output
The prepared schema and SQL view are used in Week 2 for Python-based RFM analysis.
